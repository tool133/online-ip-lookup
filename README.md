# Online IP Lookup

IP 地址查询工具专题项目，用于识别 IPv4、IPv6、私有地址、公网地址和常见网络范围。

## Online Tool

[IP地址查询工具](https://www.abplus.xin/tools/ip-lookup)

## Features

- IPv4 地址识别
- IPv6 地址识别
- 私有地址判断
- 公网地址和保留地址检查

## Use Cases

- 分析访问日志中的 IP
- 排查内网和公网地址问题
- 检查服务配置中的地址
- 运维排障和安全分析
